<p>畅销哥在使用了chatgpt之后感觉这太酷了，很符合他对未来人工智能的想象，科技并带着趣味，于是他爆肝十个甚至九个小时，做出了一个flaggpt，意在自动进行从做题到拿奖的所有工作，这是程序的第一次调试：</p>
